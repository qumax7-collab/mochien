### 4-1. 모찌엔 (일본 경제뉴스 YouTube) — 메인 트랙 [거시 전환]

**대전환 (2026-05-21)**: "당일 뉴스 매일 양산" → "거시경제 에버그린 + 반자동 제작".

전환 이유:
- 당일 뉴스 롱폼은 유통기한이 하루 → 콘텐츠가 매일 죽어 자산이 안 쌓임
- 거시 원리("왜 이렇게 돌아가는가")는 에버그린 → 1년 뒤 검색에도 유효, 누적 자산
- 반자동 제작(운영자 개입)으로 양산형 판정을 벗어나고 신뢰를 확보
  (2025-07 YouTube 정책 핵심 = "인간의 독창적 가치가 있는가")

**구조**:
- 쇼츠: 현행 유지 (매일 2개, 07:00 / 18:00 JST). 당일 뉴스 미끼 = 유입 담당.
  완전 자동 아님 — UI로 기사·배경 선택.
- 롱폼: 당일 뉴스 매일 → **거시 원리형 주 2편** (21:00 JST). 신뢰 자산 = 수익 담당.
- 역할 분담: 쇼츠=유입(자동), 롱폼=자산(반자동, 운영자 개입).

**롱폼 제작 워크플로우 (반자동)**:
1. 자료 수집 → AI가 **실제 자료에 근거하여** 한국어 거시 원리형 대본 생성
   (환각 방지: 자료에 없는 수치·주장 금지. 주요 주장마다 근거 표기)
2. 운영자가 한국어 대본 읽고 숙지 (웹 UI에서 textarea로 직접 편집 가능 — 오늘 추가)
3. 모르는 부분 → "더 쉽게 풀어줘" 대화로 재요청
4. 만족 시 일본어 변환 (모찌엔 페르소나 톤)
5. 일본어 → 한국어 역직역으로 의도 확인 (운영자가 일본어 못 하므로 안전장치)
6. 발행

**모드 A/B 자동 분기** (long1_script.py):
- 당일 기사 2건이 토픽 뱅크의 같은 원리로 묶이는지 GPT가 판정
- 모드 A (묶임): intro → issue1(원리) → issue2(같은 원리 다른 각도) → outro
- 모드 B (안 묶임, 대부분): 더 강한 1건만 골라 주제 확정 → issue1·2를 그 주제의
  원리·심화로 채움. 나머지 1건은 버리거나 intro에서 가볍게 언급.
- 핵심: 원리형은 '기사=입구'라 기사 1건만 좋아도 영상 성립.

**모찌엔 페르소나** (long1 프롬프트에 일본어로 반영):
- 경제 입문자 눈높이의 거시 교양 해설가
- 뉴스를 입구로 "왜 이렇게 되는가" 원리 설명. "왜?"로 시작.
- 전문가에겐 당연한 것도 초보자에겐 아니라는 전제로 친절히

### 5-1. 한국 쿠팡 파트너스 블로그 (2026-05-21 폐기)
- (과거) 모찌엔 다음 메인 트랙 후보였음
- 폐기 사유:
  ① 모찌엔 블로그는 일본 시장 → **쿠팡은 일본 미서비스라 작동 불가**
  ② AI 양산 블로그는 2026 구글 AI 오버뷰로 트래픽 자체가 안 붙음
  ③ 방치형이라 운영자 흥미 이탈 트리거
  ④ "빠른 수익" 목적과 안 맞음 — 트래픽 선행 필요(6개월+), 3% 수수료라
     월 30만원 = 거래액 1,000만원(하루 11건 구매) 필요
  ⑤ 쿠팡 본사 적자·탈팡·영업정지 검토 등 시점 리스크

  ① 빠르지 않음 — 전략·백테스트·실전검증에 수개월+
  ② 수익 모델이 아니라 **투자** → 실패 시 시간이 아닌 **원금 손실**
  ③ 진입장벽(통계·금융·시스템 리스크) + 작은 종잣돈으론 월 몇십만원 비현실
  ④ 트레이딩 자산은 코인으로 변환 완료 (자금 계획 충돌)
- 흥미가 있다면 여유 자금으로 천천히 하는 장기 투자로는 가능. 빠른 수익 답은 아님.
